# Docs Overview

This folder contains the working documentation for the educational database engine project.

## Main docs
- `PROJECT_CONTEXT.md` explains the purpose of the project and the current learning goals
- `PROJECT_PLAN.md` describes the implementation roadmap from storage up to higher database layers
- `STORAGE_NOTES.md` tracks the current storage design and what is actually implemented
- `CMAKE_TUTORIAL.md` explains how the build setup works
- `REFERENCES.md` lists the main study references behind the project

## Codex handoff
- `CODEX_AGENT_CONTEXT.md` is the orientation file for future Codex sessions
- if the project is opened on another device, this file can be shared with the agent first so it understands the project intent, collaboration style, and documentation expectations

## Documentation direction
The documentation in this folder should stay close to the implementation.

When storage behavior changes, `STORAGE_NOTES.md` should be updated so the current engine design stays documented clearly.
